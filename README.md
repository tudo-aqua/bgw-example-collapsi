# BGW Example: Collapsi

## Rules

Collapsi is a simple board game invented by Mark S. Ball in 2025.

The "living-rules" document of Collapsi can be
found [here](https://docs.google.com/document/d/1MV1vvV1Aq7ikRU7lF8Y7QQD-akH3ybHmk2fKs6p61lE/edit).
A copy of the rules can be found in "Collapsi - Rules.pdf".
